<task type="auto">
  <name>Task 2: Create conftest.py with shared FastF1 and telemetry fixtures</name>
  <files>tests/conftest.py</files>
  <action>
    Create `tests/conftest.py` with these fixtures:

    1. **`reset_settings_singleton`** (autouse=True) — Set `SettingsManager._instance = None` before and after each test. Import from `src.lib.settings`.

    2. **`mock_telemetry_df`** — Factory fixture returning a function that creates a small pandas DataFrame mimicking FastF1 telemetry with columns: SessionTime (timedelta), X, Y, Distance, RelativeDistance, Speed, nGear, DRS, Throttle, Brake. Use ~10 rows with realistic F1 values (speeds 200-340 kph, gears 3-8, distances 0-5000m). SessionTime should be pd.to_timedelta with incrementing seconds.

    3. **`mock_lap`** — Factory fixture returning a function `make_lap(lap_number, compound, tyre_life, telemetry_df)` that creates a MagicMock with attributes: LapNumber, Compound, TyreLife, LapTime (pd.Timedelta). The mock's `get_telemetry()` returns the provided telemetry_df.

    4. **`mock_laps`** — Factory fixture using mock_lap. Returns a MagicMock that has:
       - `.empty` = False
       - `.pick_drivers(driver_no)` that filters by driver number
       - `.iterlaps()` that yields (index, lap) tuples
       - `.LapNumber.max()` returning the highest lap number
       Build with 2 drivers (driver '1' with 3 SOFT laps, driver '44' with 3 MEDIUM laps).

    5. **`mock_session`** — Full mock FastF1 session using mock_laps. Includes:
       - `.laps` = mock_laps result
       - `.weather_data` = DataFrame with TrackTemp, AirTemp, Humidity, WindSpeed, Rainfall (3 rows)
       - `.results` = DataFrame with DriverNumber, Abbreviation, TeamName, TeamColor, Position
       - `.event` = dict-like with EventName='Test GP', Location='Test Circuit'
       - `.total_laps` = 3

    6. **`sample_telemetry_frame`** — Dict matching the TCP stream protocol format:
       ```python
       {
           "frame_index": 0,
           "total_frames": 1000,
           "playback_speed": 1.0,
           "is_paused": False,
           "circuit_length_m": 5412.0,
           "track_status": {"status": "1", "message": "AllClear"},
           "frame": {
               "drivers": {
                   "1": {"x": 100.0, "y": 200.0, "speed": 310.5, "gear": 7, ...},
                   "44": {"x": 150.0, "y": 250.0, "speed": 305.2, "gear": 6, ...}
               },
               "weather": {"TrackTemp": 45.0, "AirTemp": 28.0, ...}
           }
       }
       ```
       Include at least 2 drivers with all fields: x, y, speed, gear, drs, throttle, brake, lap, tyre_compound, tyre_life, relative_distance.

    7. **`tmp_settings`** — Fixture that patches `SettingsManager._get_settings_file_path` to return a path inside `tmp_path`, so settings tests use isolated temp files.

    Use `from unittest.mock import MagicMock, patch` — do NOT import pytest-mock's mocker here (that's per-test). conftest fixtures use standard unittest.mock.
