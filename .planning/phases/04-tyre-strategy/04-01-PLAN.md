Create a `StintAccumulator` class (following the LapTimeAccumulator pattern from lap_time_evolution_window.py) that:

1. **Stint dataclass:**
   ```
   @dataclass
   class Stint:
       driver: str
       compound_int: int      # 0-4
       start_lap: int         # first lap on this compound
       end_lap: int | None    # None if stint is ongoing
       tyre_life_start: int   # tyre_life at stint start
   ```

2. **StintAccumulator class:**
   - `self.stints: dict[str, list[Stint]]` — driver_code -> list of completed + current stints
   - `self._driver_states: dict[str, _StintDriverState]` — tracking current_lap, current_compound, current_stint
   - `process_frame(data: dict)` method:
     - For each driver in frame: track current lap and compound
     - When compound changes between frames → close current stint (set end_lap), start new stint
     - When lap increases → update current stint's end_lap to current lap
     - Initialize first stint on first frame seen for each driver
   - `get_stints(code: str) -> list[Stint]` — return stints for a driver
   - `get_all_drivers() -> list[str]` — return sorted list of all drivers seen
   - `get_max_lap() -> int` — return maximum lap number seen across all drivers

3. **_StintDriverState dataclass:**
   ```
   @dataclass
   class _StintDriverState:
       current_lap: int | None = None
       current_compound: int | None = None
   ```

Key behavior:
- First frame for a driver starts stint 1
- Compound change → new stint (close old, start new)
- Ongoing stint has end_lap = current_lap (updated every frame)
- Handle None compound gracefully (skip frame)
